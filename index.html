Reasearch-Project
[Brain-Computer-Interface]

Optogenetics-Neurotechnology

Optogenetics is an emerging technology for controlling or changing behaviour in animal by using genetic-material. In this project- Optogenetic Stimulation System (4-Channel Controller), Compatible Commutators, Opto-electrode, LED Modules, Linear Probes, and Light Measurement Kit are used.

Mindwave-Mobile-EEG-Headset"

EEG is a technique for recording and analysing neurocognitive activities of brain. In this project- Neurosky-Mindwave-Mobile-Headset (Single Channel), Brainwave-Visualizer, Blink-Camera, Speed-Math, and Android-Developer Tool are used.

Human-Memory-Consolidation

Memory Consolidation is the technique of converting short-term memory into long-term memory. In this project fMRI scanner used for neuroimaging from neocortical regions of the brain.
